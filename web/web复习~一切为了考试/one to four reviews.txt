 		<p class="txt">
 			<ol>
 				<h4>网页浏览过程</h4>
 				<li>通过输入网址（URL）指定要访问的网页</li>
 				<li>请求：把XXX网页文件传送给我</li>

 			</ol>
 			<h4>网页文件</h4>
 			<img src="1~4图片/网页文件.png" alt="网页文件"/>
 			浏览器中看到的网页实质为：网页文件<br/>
 			<ul>网页文件
 				<li>文本文件</li>
 				<li>扩展名.html或.htm</li>
 				<li>文件内容HTML代码和文本内容</li>
 			</ul>
 			<h4>HTML简介</h4>
 			<ul>
 				<li>HTML:Hyper Text Markup Language</li>
 				超文本标记语言
 				<li>HTML不是一种编程语言，而是一种标记语言。标记语言是一套<br/>标签，HTML使用标记来描述网页~</li>
 			</ul>
 			<h4>HTML标记标签通常被称为HTML标签，它是由尖括号包围的关键词。</h4>
 			<h4>HTML基本结构</h4>
 			<img src="1~4图片/基本结构.png" alt="基本结构"/>
 			<ul>
 				<h4>标签的分类：</h4>
 				<li>双标签：由“开始标签”和“结束标签”两部分构成。<br/>
 				结束标签比开始标签多一个“/”（比如《body》《/body》）,必须成对使用。<br/>
 					<h6>双标签：代表标签作用范围</h6>
 				</li>
 				<li>单标签：在开始标签中进行关闭，即以开始标签的结束而结束（比如《meta/》）<br/>
 					<h6>单标签：无需表达范围，仅在标签出现出有效</h6></li>
 			</ul>
 			<h4>HTML元素指的是从开始标签到结束标签的所有代码</h4>
 			<h4>标签书写规范:</h4>
 			<ul>
 				<li>标签与标签之间可以嵌套的，但先后书写序必须保持一致。如：<br/>
 				《div》《p》小雨不萌《/p》《/div》</li>
 				<li>HTML标签对大小写不敏感，但建议同一规范小写</li>
 			</ul>
 			<h4>HTML标签可以拥有属性——辅助标签提供更多信息</h4>
 			<ul>
 				<li>属性书写形式：name="value"</li>
 				<li>属性书写位置：开始标签</li>
 				<li>不同标签具有不同属性</li>
 				<li>
 					<ul>
 						<li>bgcolor		颜色名称		规定文档的背景颜色</li>
 						<li>text     	颜色名称		规定文档中所有文本颜色</li>
 					</ul>
 				</li>
 			</ul>
 			<h4>图片路径：</h4>
 			<ul>
 				<li>
 					绝对路径：
 					<ul>
 						<li>优点：真实路径，定位清晰</li>
 						<li>缺点：本机绝对路经长，容易出错<br/>
 							如果站点文件夹被移动，就需要重新修改路径<br/>
 							<h6>不推荐使用本机绝对路径</h6>
 						</li>
 					</ul>
 				</li>
 				<li>
 					相对路径：
 					<ul>
 						<li>相对路径是指相对当前文件后目录的路径</li>
 						<li>相对路径规则：
 							<ol>
 								<li>图片和网页在同级目录	src="./图片名"</li>
 								<li>图片在网页的下一级目录	src="目录名称/图片名"</li>
 								<li>图片在网页的上一级目录 src="../图片名"</li>
 							</ol></li>
 						<li>优点：文件夹被移动，其内部文件的相对路径不变</li>
 					</ul>
 				</li>
 				<li>注意事项：
 					<ul>
 						<li>不要使用本机绝对路径，推荐使用相对路径</li>
 						<li>图片文件单独存放在一个文件夹中</li>
 						<li>图片文件夹与页面文件放在同一个目录下</li>
 					</ul></li>
 			</ul>
 			<ul>
 				<h4>超链接——从一个网页指向一个目标的链接关系</h4>
 				<li>《a href="链接目标"》链接对象《/a》</li>
 				<li>适合用来制作超链接的内容：导航、推荐图文、友情链接等</li>
 				<li>链接目标：另一个网页、图片等</li>
 				<li>必选属性：href,创建指向另一个文档的链接等</li>
 				<h5>属性：</h5>
 					<li>href:规定链接目标（链接的而目标URL）</li>
 					<li>target:在何处打开目标
 						<ol>
 							<li>-blank:在新窗口打开</li>
 							<li>-self:在当前窗口打开（默认）</li>
 						</ol>
 					</li>
 			</ul>
 			<ul>
 				列表——HTML支持无序列表、有序列表
 				<li>无序列表：是一个没有前后顺序的信息列表<br/>
 					<ul>
 						<li>无序列表使用《ul》标签。每个列表项使用《li》标签</li>
 						<li>每个列表项默认使用粗体远点进行标记</li>
 						<li>列表项内部可以使用段落、换行符、图片、链接以及其他列表等</li>
 					</ul></li>
 				<li>有序列表：是一个有前后顺序的信息列表<br/>
 					<ul>
 						<li>有序列表使用《ol》标签</li>
 					</ul></li>
 			</ul>
 			<h4>注意事项与编码规范</h4>
 			<ul>
 				注意事项：<br/>
 				<li>文件扩展名以.html或.htm结束</li>
 				<li>文件名由英文字母、数字或下划线组成</li>
 				<li>文件名中不要包括特殊符号，比如空格、￥等，不使用中文</li>
 				编码规范：<br/>
 				<li>所有的标签必须关闭</li>
 				<li>所有的标签和其属性的名字都必须使用小写</li>
 				<li>所有的标签都必须合理嵌套</li>
 				<li>所有属性必须赋值，所有属性值必须用引号括起来</li>
 			</ul>
 			</p>
 		</ol>

		<h1 class="title">第四章 CSS语法基础</h1>
		<p class="txt">
			<ul>
			<h4>CSS概述：</h4>
			<li>CSS是Cascading Style Sheet的缩写【层叠样式表】 </li>
			<li>CSS是用于（增强）控制网页样式并允许将样式与网页内容分离的一种标记性语言</li>
			<h4>为什么使用CSS</h4>
			<li>
				<ol>单纯HTML方法控制元素样式的缺点<br/>
					<li>针对元素样式设置的属性太少</li>
					<li>修改元素样式太麻烦</li>
					<li>控制元素样式的代码*余度过高</li>
				</ol>
			</li>
			<li>
				<ol>使用CSS的优点</ol>
				<li>更专业的样式修饰方法</li>
				<li>改一处动全局的方法</li>
				<li>更简便的而样式修饰方法</li>
				<li>更简约的而不惧方法</li>
			</li>
			<h4>CSS语法</h4>
			CSS代码是由一条条语句构成，而每一条语句的结构，都基本相同<br/>
			选择器{属性：属性值；属性：属性值；.....}<br/>
			<img src="1~4图片.png/CSS语法.png"/>
			<ul>
				<h4>选择器：规定该选择器定义的样式将对那些元素生效</h4>
				<li>
					标签选择器——选择器是HTML标签<br/>
					<img src="1~4图片/标签选择器.png"/>
				</li>
				<li>
					类选择器——以“.”开头定义的选择器<br/>
					<img src="1~4图片/类选择器.png"/>
					<h6>注意：类名的第一个字符不能使用数字<br/>
						元素可以加入多个类。把各个类的名放在class属性中，各个类<br/>名之间用一个空格隔开，类名的额顺序并不重要。</h6>
				</li>
				<li>
					ID选择器——以“#”开头定义的选择器<br/>
					<img src="1~4图片/ID选择器.png"/>
				</li>
			</ul>
			<li>行内样式——在元素标签内通过style属性添加样式
				<br/>
			用途：指定网页中个别元素的而现实小狗。不符合样式与内容分离原则，不推荐使用<br/>
			<img src="1~4图片/行内样式.png"/></li>
			最灵活，但不符合样式与内容分离原则，不推荐使用。
			</ul>
			<li>
				页内样式——在head部分的style标签内添加样式<br/>
				用途：对页面中某些标签或元素设置样式风格。控制当前页面样 式，维护较困难。<br/>
				控制单一页面，维护较困难，不推荐使用。
			</li>
			<li>
				外部样式 —— 引用外部建立的.css文件 <br/>
				用途：可同时控制多个页面，适用于各类大型网站，可用性最强， 推荐使用。<br/>
				<img src="1~4图片/外部样式.png"/>
				• 可用性最强，可同时控制多个页面，推荐使用 。
			</li>
