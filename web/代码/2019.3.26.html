css

css是Cascading style sheet 的缩写（层叠样式列表）
css是用于（增强）控制网页的样式并允许样式与网页内容
分离的一种标记性语言

网页内容是通过html来做的，而样式是通过css完成的


单纯html方法控制元素样式的缺点

  • 针对元素设置的属性太少
  • 修改元素样式太麻烦
  • 控制元素样式的代码*余度过高

css代码

ul li a { color:red; font-size:20px; }
  • 针对元素设置的属性太少
  • 修改元素样式太麻烦
  • 控制元素样式的代码*余度过高

选择器

选择器：规定选择器定义的样式对那些元素生效
标签选择器——选择器是HTML标签

  1. 选择器
  2. 大括号
  3. 属性：属性取值（属性之间用；分隔）

类选择器——以”.“开头定义的选择器
影响了所有以class属性引用该类的标签样式
注意：类名的第一个字符不能使用数字
noie,非常棒的，么么哒·~

id选择器——以“#”开头定义的选择器 影响以ID属性引用该选择器的标签样式 按需设置
注意：单一页面中，一个id选择器只能使用一次！

  • 如果希望一个元素加入多个类
  • 各个类名之间用一个空格分隔
  • 类名的顺序并不重要

插入样式表的三种方式：

  • 行内样式
  • 页内样式
  • 外部样式


    • 行内样式——在元素标签通过style属性添加样式
    <标签 style="属性：属性值；属性：属性值；。。。。">

    并不推荐使用行内样式

    页内样式——在head部分的style标签内添加样式
    p{}
    .*{}
    #*{}
    用途：对页面中某些标签或元素设置样式风格

    外部样式文件——引用外部建立的.css文件 可以控制多个，推荐使用
